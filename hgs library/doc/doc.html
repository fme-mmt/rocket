HGS. Chemical Equilibrium Calculation

HGS aims to provide an open source and comprehensive approach to rocket combustion problems and further isentropic expansion using the NASA polynomial approach.

Full version of this code can be downloaded from Matlab File Exchange.

To work properly, the code requires BurcatDB.mat to be on the same folder. This file contains all data from the NASA polynomial to compute gas properties.

For further reference about the code see the documentation attached or email at: open(dot)llop(at)gmail(dot)com

Functions

  • hgseq: This function computes the chemical equilibrium of species.
  • hgsisentropic: Computes an isentropic expansion assuming either frozen or shifting flow.
  • hgsprop: This function computes function computes properties for a single element or mixtures whose temperatures range from 200 K to 6000 K.
  • hgssingle: This function computes the enthalpy, entropy or Gibbs free energy of a single element. The temperature must be set within a range from 200 K to 6000 K.
  • hgsTp: Computes the adiabatic flame temperature of a reaction (with dissociation).
